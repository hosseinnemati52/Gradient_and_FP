This is solving the problem by stochastic simulation.
2d rectangular lattice graph.
Periodic boundary conditions in vertical direction,
 and open boundary conditions in horizontal direction 
 (without self-loops)
Number of rows and columns determined in the file N_xy.csv. (The first one is the number of rows)
In the file r_s_data.csv, the numbers are [rA, rB, s]; rA=inherent fitness of resident; rB=inherent fitness of mutant; s=gradient parameter of the fitness of mutant;
The file fix_probs_blocks.csv shows different samples for the fixation probability of the mutant.
